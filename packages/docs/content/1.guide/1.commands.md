## Command

### Options

```bash
npx vedix -h
```

#### Init

```bash
npx vedix init
```
Create a vedix.config.json file at the prompts

```json
{
  "components": "2.components/ui"
}
```

#### Add

Create a component

```bash
npx vedix add
```

###### Components

- **AspectRatio**
- **Avatar**
- **Button**
- **Label**
- **Progress**
